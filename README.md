Este projeto foi desenvolvido para gerenciar e analisar dados de dispositivos Apple em uma loja fictícia chamada TechStore.  
Os dados são armazenados em um banco de dados SQL, manipulados com Pandas em Python e exportados para planilhas Excel com os relatórios necessários.

- Conexão com banco de dados SQL contendo dispositivos Apple.
- Consulta e manipulação dos dados com Pandas.
- Exportação dos resultados para Excel.
- Possibilidade de gerar relatórios como:
    Estoque atual de dispositivos.
    Vendas realizadas por modelo.
    Comparação de preços entre dispositivos.
    Ranking dos dispositivos mais vendidos.
    Adicionar / remover produto diretamente com Python.
